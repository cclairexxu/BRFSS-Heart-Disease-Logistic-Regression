# Multivariable Logistic Regression Analysis of Diabetes and Modifiable Risk Factors for Heart Disease: 2022 BRFSS Data

## Description
Analyzed 246,000+ U.S. adult records from the 2022 CDC BRFSS dataset to examine diabetes and modifiable risk factors for heart disease. Conducted multivariable logistic regression, odds ratio estimation, and ROC/AUC analysis in R. Secondary analyses explored physical inactivity, smoking, and self-reported health as predictors. Report written in Quarto for full reproducibility.
